Interactive prototypes exploring how property-level marketers create personalized guest communications using a shared content library. Select a use case to begin.

Use Case 1
Creating a New Email Version

Mary wants to create a pre-arrival email tailored to group business travelers. Instead of building from scratch, she answers two questions — and the system assembles the right content from her library automatically. The same library produces a completely different email for a leisure traveler.

Use Case 2
Building the Amenity Library

Mary creates a new amenity — giving it a name, a description (with AI assistance), and conditions for who should see it and when. Once saved, it lives in the library and surfaces automatically across channels. The same amenity can power the pre-arrival email and the lobby screen without duplication.
